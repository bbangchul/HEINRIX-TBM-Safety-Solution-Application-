# HEINRIX TBM Safety Solution Application

건설 현장 TBM(Tool Box Meeting) 운영을 디지털화한 모바일 안전관리 솔루션입니다.

## Overview
HEINRIX는 현장 안전회의의 준비부터 결과 산출까지 한 번에 처리할 수 있도록 설계된 앱입니다.

- 프로젝트/참여 인원 권한 관리
- 현장 참여 요청 및 승인 프로세스
- 근로자 참석 및 전자서명 관리
- 현장 사진 등록
- TBM 보고서(PDF) 자동 생성 및 문서 이력 관리

## Why HEINRIX
- 수기 TBM 문서 작업 최소화
- 권한 기반 운영으로 데이터 정합성 강화
- 현장/본사/협력업체 간 커뮤니케이션 효율 향상

## Tech Stack
- Flutter
- Firebase Auth / Firestore / Cloud Functions / Storage
- TypeScript (Cloud Functions)
